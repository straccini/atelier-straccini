Guida Apps Script

1. Vai su script.google.com → Nuovo progetto

2. Incolla il codice Code.gs (scaricato separatamente)

3. Sostituisci SHEET_ID con l'ID del tuo Google Sheet

4. Distribuisci → App Web → Accesso: Chiunque

5. Copia l'URL e incollalo nel campo qui sopra

6. Condividi l'URL dell'app (questo file HTML) con i collaboratori
